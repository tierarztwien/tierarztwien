## Technische Anforderungen

### Fuer den Fotografen
- **Format:** RAW + JPEG (hoechste Qualitaet)
- **Ausrichtung:** Sowohl Querformat (Website-Header) als auch Hochformat (Mobile, Instagram)
- **Farbstimmung:** Warm, natuerlich, passend zum bestehenden Bildstil (cremig, golden)
- **Mindestaufloesung:** 3000px Langseite
- **Lieferung:** Innerhalb 7 Tage, bearbeitet + unbearbeitet

### Bildbearbeitung (Post-Production)
- Farbkorrektur passend zum Website-Farbschema (warm, cream/gold-Toene)
- Leichte Hautretusche (natuerlich belassen)
- Zuschnitte fuer verschiedene Verwendungen:
  - Website Hero: 1920x1080px (16:9)
  - Galerie quadratisch: 1200x1200px (1:1)
  - Instagram: 1080x1080px (1:1) und 1080x1350px (4:5)
  - Google Business: 720x720px (1:1)
  - Portrait: 800x1000px (4:5)

### SEO-Dateibenennung
Alle Fotos nach diesem Schema benennen:
```
tierarzt-[motiv]-[detail]-wien.jpg
```
Beispiele:
- `tierarzt-antolini-portrait-wien-hietzing.jpg`
- `tierarzt-katze-untersuchung-wien.jpg`
- `villa-ordination-aussenansicht-hietzing.jpg`
